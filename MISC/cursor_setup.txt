================================================================================
                         FILE FORMAT REFERENCE
================================================================================

Every .mdc file must start with YAML frontmatter:

---
description: Short description shown in Cursor's rule picker
globs: optional/path/pattern/**/*
alwaysApply: false
---

# Title

Content with markdown formatting...

KEY METADATA FIELDS:
- description: Shows up when Cursor lists available rules
- globs: If set, rule only applies when editing files matching pattern
- alwaysApply: true = ALWAYS injected, false = AI-requested when relevant
